\section{Обмен данных с веб-странице}
\label{dataexchange}

\subsubsection{Название странице}

Название страницы можно получить вызывая \mintinline{icl}{[r/*] Tab'title : string}.

\subsubsection{Исходный код}

Исходный код страницы можно получить вызывая \mintinline{icl}{[r/o] Tab'source : string}.

\subsubsection{Screenshot}

Screenshot страницы в base64 можно получить вызывая\\* \mintinline{icl}{[r/o] Tab'screenshot : string}.

\subsubsection{URL}

Адрес доступен через свойство \mintinline{icl}{[r/w] Tab'url : string}.

\subsubsection{Навигация}

\mintinline{icl}{Tab.back () : void} — позволяет возвращаться на предыдущую страницу.

\mintinline{icl}{Tab.forward () : void} — позволяет возвращаться на следующую страницу.

\mintinline{icl}{Tab.refresh () : void} —  позволяет перезагрузить страницу.

\mintinline{icl}{[icL] [r/o] Tab'canGoBack : bool} — сообщает можно ли перейти на предыдущую страницу.

\mintinline{icl}{[icL] [r/o] Tab'canGoForward : bool} — сообщает можно ли перейти на следующую страницу.


\subsubsection{Предупреждения}

\mintinline{icl}{[r/o] Alert'text : string} — текст предупреждений.

\mintinline{icl}{Alert.accept () : void} — закрыть окно отвечая положительным ответом.

\mintinline{icl}{Alert.dismiss () : void} — закрыть окно отвечая отрицательным ответом.

\mintinline{icl}{Alert.sendKeys (str : string) : void} — ответить текстом \mintinline{icl}{str}.
