\section{Типы данных}

В данном главе представлены типы данных, которые могут быть r-значениями, l-значениями и js-значениями. Все типы отсортированы в алфавитном порядке, только первое место занимает \code{value}.

\subsection{{\color{lightblue} value}}

\code{value} предлагается в роли базового класса остальных типов описанных в данном главе.

\noindent Описание класса \code{value} -
\begin{lstlisting}[numbers=none]
class value {
    value'lValue   : bool;
    value'link     : bool;
    value'rValue   : bool;
    value'readOnly : bool;
    value'typeId   : int;
    value'typeName : string;

    value.ensureRValue () : value;
}
\end{lstlisting}
