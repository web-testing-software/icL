\section{Comenzi adiționale}

Limbajul de scriptare icL conține următoarele comenzi adiționale:
\begin{icItems}
	\item \mintinline{icL}{Sessions : sessions};
	\item \mintinline{icL}{Session : session};
	\item \mintinline{icL}{Windows : windows};
	\item \mintinline{icL}{Window : window};
	\item \mintinline{icL}{Cookies : cookies};
	\item \mintinline{icL}{Alert : alert};
	\item \mintinline{icL}{Tabs : tabs};
	\item \mintinline{icL}{Tab : tab};
	\item \mintinline{icL}{Document : document}.
\end{icItems}

\subsubsection{\mintinline{icL}{Sessions : sessions}}

Returnează lista de sesiuni.

\subsubsection{\mintinline{icL}{Session : session}}

Returnează sesiunea curentă.

\subsubsection{\mintinline{icL}{Windows : windows}}

Returnează lista de ferestre în sesiunea curentă.

\subsubsection{\mintinline{icL}{Window : window}}

Returnează fereastra tabului activ.

\subsubsection{\mintinline{icL}{Cookies : cookies}}

Returnează lista fișierelor cookie ale tabului curent.

\subsubsection{\mintinline{icL}{Alert : alert}}

Returnează alerta curentă a sesiunii curente.

\subsubsection{\mintinline{icL}{Tabs : tabs}}

Returnează lista de taburi a sesiunii curente.

\subsubsection{\mintinline{icL}{Tab : tab}}

Returnează tabul curent.

\subsubsection{\mintinline{icL}{Document : document}}

Returnează documentul tabului curent.
