\section{Timp}

Marca de timp (tipul \mintinline{icl}{datetime}) nu poate fi descrisă prin literal, dar poate fi creată din șir de caracter cu format specific.

\subsection{Operatori}

Pentru mărci de date sunt disponibili următorii operatori de rangul 1:
\begin{icItems}
	\item \mintinline{icl}{string : datetime};
	\item \mintinline{icl}{(dt : string) : (datetime, format : string)};
	\item \mintinline{icl}{datetime : string};
	\item \mintinline{icl}{datetime : (string, format : string)};
\end{icItems}

\subsubsection{\mintinline{icl}{string : datetime}}

Conversează șirul de caractere în \mintinline{icl}{datetime}, folosind formatul \\* \mintinline{icl}{yyyy-MM-ddTHH:mm:ss[Z|[+|-]HH:mm]}.

\subsubsection{\mintinline{icl}{(dt : string) : (datetime, format : string)}}

Conversează șirul de caractere \mintinline{icl}{dt} în \mintinline{icl}{datetime}, folosind formatul indicat, exemplu: \mintinline{icl}{"21.01.2019" : (datetime, "dd.MM.yyyy")}. Lista placeholder-elor disponibile:
\begin{icItems}
	\item \mintinline{icl}{d} - data;
	\item \mintinline{icl}{dd} - data cu zero înainte;
	\item \mintinline{icl}{ddd} - prescurtarea denumirii zilei;
	\item \mintinline{icl}{dddd} - denumirea completă a zilei;
	\item \mintinline{icl}{M} - luna;
	\item \mintinline{icl}{MM} - luna cu zero înainte;
	\item \mintinline{icl}{MMM} - prescurtarea denumirii lunii;
	\item \mintinline{icl}{MMMM} - denumirea completă a lunii;
	\item \mintinline{icl}{yy} - anul în 2 cifre;
	\item \mintinline{icl}{yyyy} - anul în 4 cifre;
	\item \mintinline{icl}{h} - ora;
	\item \mintinline{icl}{hh} - ora cu zero înainte;
	\item \mintinline{icl}{m} - minuta;
	\item \mintinline{icl}{mm} - minuta cu zero înainte;
	\item \mintinline{icl}{s} - secunda;
	\item \mintinline{icl}{ss} - secunda cu zero înainte;
	\item \mintinline{icl}{ap} - am ori pm;
	\item \mintinline{icl}{AP} - AP ori PM;
\end{icItems}

\subsubsection{\mintinline{icl}{datetime : string}}

Conversează marca de timp în șir de caractere cu formatul \\*\mintinline{icl}{yyyy-MM-ddTHH:mm:ss[Z|[+|-]HH:mm]}.

\subsubsection{\mintinline{icl}{datetime : (string, format : string)}}

Conversează marca de timp în format indicat.

\subsection{Properties}

Marca de timp are următoarele proprietăți:
\begin{icItems}
	\item \mintinline{icl}{[r/w] datetime'year : int};
	\item \mintinline{icl}{[r/w] datetime'month : int};
	\item \mintinline{icl}{[r/w] datetime'day : int};
	\item \mintinline{icl}{[r/w] datetime'hour : int};
	\item \mintinline{icl}{[r/w] datetime'minute : int};
	\item \mintinline{icl}{[r/w] datetime'second : int};
	\item \mintinline{icl}{[r/o] datetime'valid : bool}.
\end{icItems}

\subsubsection{\mintinline{icl}{[r/w] datetime'year : int}}

Anul.

\subsubsection{\mintinline{icl}{[r/w] datetime'month : int}}

Luna.

\subsubsection{\mintinline{icl}{[r/w] datetime'day : int}}

Data.

\subsubsection{\mintinline{icl}{[r/w] datetime'hour : int}}

Ora.

\subsubsection{\mintinline{icl}{[r/w] datetime'minute : int}}

Minuta.

\subsubsection{\mintinline{icl}{[r/w] datetime'second : int}}

Secunda.

\subsubsection{\mintinline{icl}{[r/o] datetime'valid : bool}}

\true, dacă marca de timp e corectă, în caz contrar \false.

\subsection{Methods}

Marca de timp are următoarele metode:
\begin{icItems}
	\item \mintinline{icl}{datetime.addSecs (secs : int) : datetime};
	\item \mintinline{icl}{datetime.addDays (days : int) : datetime};
	\item \mintinline{icl}{datetime.addMonths (months : int) : datetime};
	\item \mintinline{icl}{datetime.addYears (months : int) : datetime};
	\item \mintinline{icl}{datetime.secsTo (dt : datetime) : int};
	\item \mintinline{icl}{datetime.daysTo (dt : datetime) : int};
	\item \mintinline{icl}{datetime.toUTC () : datetime};
	\item \mintinline{icl}{datetime.toTimeZone (hours : int, minutes : int) : datetime}.
\end{icItems}

\subsubsection{\mintinline{icl}{datetime.addSecs (secs : int) : datetime}}

Adaugă numărul necesar de secunde.

\subsubsection{\mintinline{icl}{datetime.addDays (days : int) : datetime}}

Adaugă numărul necesar de zile.

\subsubsection{\mintinline{icl}{datetime.addMonths (months : int) : datetime}}

Adaugă numărul necesar de luni.

\subsubsection{\mintinline{icl}{datetime.addYears (months : int) : datetime}}

Adaugă numărul necesar de ani.

\subsubsection{\mintinline{icl}{datetime.secsTo (dt : datetime) : int}}

Calculează diferența de timp în secunde.

\subsubsection{\mintinline{icl}{datetime.daysTo (dt : datetime) : int}}

Calculează diferența de timp în zile.

\subsubsection{\mintinline{icl}{datetime.toUTC () : datetime}}

Returnează o nouă marcă de timp, care conține timpul UTC.

\subsubsection{\mintinline{icl}{datetime.toTimeZone (hours : int, minutes : int) : datetime}}

Returnează o nouă marcă de timp, care conține timpul în fusul orar cerut.

\subsection{Timpul curent}

Sunt disponibile următoarele metode de a primi timpul curent:
\begin{icItems}
	\item \mintinline{icl}{Datetime.current () : datetime};
	\item \mintinline{icl}{Datetime.currentUTC () : datetime}.
\end{icItems}

\subsubsection{\mintinline{icl}{Datetime.current () : datetime}}

Timpul curent local.

\subsubsection{\mintinline{icl}{Datetime.currentUTC () : datetime}}

Timpul curent UTC.
